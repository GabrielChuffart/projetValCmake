Les library sfml ajoutées n'étant pas celles de debug, le projet doit nécéssérement être compilé en "release" et non en "debug".
Nous avons préalablement configuré une version release avec cmake, son fichier .json étant présent sur le dépot.
Toute fois, si la version release n'était pas reconnue, l'ajout manuel d'une version release sera requis. 
Nous nous excusons par avance si cela devient une nécéssité.
